- Créer un tableau d'objet appareils  
- Ajouter un appareils    
- Allumer / Eteindre tous les appareils 

```js
const appareils =[
    {name:'TV', status:'allumé'},
    {name:'Xbox', status:'éteint'},
    {name:'Machine à café', status:'allumé'}
];
```
